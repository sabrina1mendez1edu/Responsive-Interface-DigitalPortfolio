# Responsive Interface Digital Portfolio

## Project Description
I created a single-page responsive digital portfolio using HTML and CSS. This page introduces who I am, highlights my creative interests, my portfolio inspiration, and photography I have done. 

## Intended User or Player
This interface is designed for anyone interested in my creative work of my digital portfolio. This includes employers, classmates, friends, and anyone looking to network.

## User / Player Goal
Users will be able to learn about me in my digital portfolio. It helps them understand my creative interests, view my photography skills, all through an easy to use interface.

## Inspiration Interfaces
I used Pinterest for visual inspiration and mood boards. I also looked at photographer portfolio websites to help guide my layout and organization.

## Design Choices
- Layout: Organized into clear sections using cards and links.
- Color: Cream, light blue, and coffee brown.
- Typography: Georgia font for a clean appearance and easy to read.
- Visual hierarchy: I used large headings, cards, and spacing to help important information stand out for my visual hierarchy.
- Feedback states: Navigation links change color when the link is hovered over by the user.
- Responsive design: Navigation stacks vertically on smaller screens using media query, making my interface easier to read for those on mobile devices.
